Rodar a aplicação em uma porta especifica
<ng serve --port 3000>

Rodar servidor do banco de dados
<json-server db.json>

Instalar na ordem que aparece
<npm install>: Sempre que ocorrer oerro: Could not find module "@angular-devkit/build-angular.

<npm install --only=dev bootstrap>: Instala o bootstrap 4
<npm install --only=dev normalize.css>: Reseta as configurações de estilo do navegador
<npm install -g rxjs>: RXJS para chamadas a backend
<npm install -g json-server>: Instalar JSON Server para Mockar o Backend
<npm install highcharts --save> Para implementar gráficos
<npm install angular-highcharts --save> Para implementar gráficos
<npm install file-saver --save> Baixar o arquivo .zip

Debug {
    insira a tag <debbuger;> no component e o debug do browser ira iniciar
}
