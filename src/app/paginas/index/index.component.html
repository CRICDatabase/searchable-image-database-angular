CRIC Searchable Image Database

This is the playground. You can login using the "Sign Up" option. It will remain active until midnight. Visit database.cric.com.br for the oficial data.

Center for Recognition and Inspection of Cells (CRIC) Searchable Image Database is a public cell image database. We have collected hundreds cell images and hand labeled classify thousants of cells, visit our Classification page to see the images and their classifications.

Access the playground to create a personal database and manipulate classifications and segmentations in the cells.

We hope CRIC Searchable Image Database will become a useful resource for cell investigation and help physicians to provide a healthier life for people worldwide.
