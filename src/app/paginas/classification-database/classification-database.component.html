Classification

This page will provide information about image classification of cervical cells in a whole image. Classification was designed by three cytopathologists. Click image to zoom and have access to cell classification in image. Each image has annotations for each nuclei which are represented by bounding boxes (100x100), all annotations and classification was selected and confirmed by three cytopathologists.
