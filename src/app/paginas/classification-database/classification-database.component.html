Cervix Collection: Classification

This is the playground. Visit database.cric.com.br for the more data.

The images represent the cytopathology of the uterine cervix using the conventional smear (Pap smear). The epithelial cell abnormalities (lesion) are addressed in six classes according to Bethesda system.

Click the image to zoom and have access to cell classification. Each image has annotations for each nucleus which are represented by bounding boxes (100pixels x 100pixels).
